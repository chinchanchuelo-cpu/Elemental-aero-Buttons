01

Introducción

Aero Elemental Buttons es una colección de botones CSS premium con animaciones, efectos de brillo, reflejos Aero Glass, gradientes multicapa y variantes temáticas. Inspirado en la estética Windows Aero, Crystal UI y efectos elementales de la naturaleza.

Para usar, incluye el archivo style.css y aplica la clase base .btn-aero-fire junto a las clases modificadoras que necesites.

05

Toggle

Haz click para cambiar estado
/* Estado OFF (por defecto) */ <button class="btn-aero-fire btn-toggle">Toggle OFF</button> /* Estado ON — añade clase .on */ <button class="btn-aero-fire btn-toggle on">Toggle ON</button> /* Toggle interactivo con JS */ btn.addEventListener('click', () => btn.classList.toggle('on'));
